Adje

an eratosthenean spectrum generator

The design of Adje is based on Ad, but it’s not an oscillator and doesn’t produce an audio signal. Instead it generates (making use of VCV Rack’s polyphony feature) two polyphonic control voltages: V/octave and amplitude. The intended use is to patch the V/octave output to an oscillator, the amps output to the CV input of a VCA, and the oscillator trough the VCA. (Both these modules should be able to work with polyphony.)

It only takes monophonic input CVs. The number of ouput channels can be set using the context menu.

For the rest, it works similar to Ad, though some knob ranges are different. And Adje doesn’t have an FM input.

There is an expander module Bufke for Adje.
